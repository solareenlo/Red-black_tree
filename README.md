# Red-black_tree
Re-implementation of Red-black tree using c++98.

## Tree Traversals
- Inorder (Left, Root, Right)
- Preorder (Root, Left, Right)
- Postorder (Left, Right, Root)

## How to implement
- Recursive
- Non recursive
- Root must be black.
- Root may be red or black.
- Left-Leaning Red-Black Tree
